# 🏥 Healthcare Web Application

A full-stack healthcare management system where patients can register, login, and book appointments with doctors.

---

## 🚀 Live Application

- 🌐 Frontend (Vercel): https://healthcare-web-application-kohl.vercel.app  
- ⚙️ Backend (Render): https://healthcare-web-application.onrender.com  

---

## 📌 Project Overview

This project is developed as part of **CSE 616 – Web Engineering Lab**.

The system allows:
- User registration and login
- Role-based access (Patient, Doctor, Admin)
- Appointment booking and management
- Secure authentication using JWT

The application is fully deployed and accessible online.

---

## 🧱 Tech Stack

### Frontend
- React.js (Vite)
- HTML, CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Authentication
- JSON Web Token (JWT)
- Bcrypt (password hashing)
